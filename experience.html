Shop application

The purpose of shop application is to allow customers to purchase products use an RFID reader. The idea of using an RFID reader is to purchase something the customer should do a check-in of his card at the beginning, then the application verifies card id in the database and shows the available amount of money. In this way, the seller doesnâ€™t spend time counting cash what makes the process faster and prevents mistakes. For the young customers, the application will automatically impose restriction for purchasing certain products which depend on constraints, for example, alcohol drinks. To keep prices and available products constantly updated at each start the application connects to the database through the internet. From the database, the application requests necessary information and based on it the product list is automatically updated. This functionality allows doing updates without changing the source code of the application what makes application maintenance easier.

C#, MySQL
Traffic simulation

Traffic simulator is application which simulates traffic jams on crossings. The main function of the program is to simulate traffic based on the inputs defined by the user.
The simulation of traffic is an aggregate of many simulations of vehicles making their way from user-defined origins to user-defined and possible destinations upon the user-defined grid. The user is also given the option to override the defaulting directions vehicles take at any particular intersection for the purpose of a more specific simulation. At the end of the simulation, the user may see statistics of traffic simulation.

C#
